# Página Web de Cumpleaños con Temática Japonesa 🎉🇯🇵

Esta es una página web creada como regalo de cumpleaños con temática japonesa, que incluye un mensaje especial y un pequeño jueguito interactivo de "Atrapa el Sushi" usando emojis.

---

## Características

- **Mensaje de cumpleaños personalizado:** Un texto bonito para felicitar a la persona especial.
- **Diseño limpio y moderno:** Usando colores suaves y estilo minimalista inspirado en Japón.
- **Juego interactivo:** Atrapa el sushi (🍣) que aparece en pantalla haciendo clic antes de que desaparezca.
- **Botón para abrir el juego:** Solo se muestra el juego cuando el usuario lo desee.

---

## Tecnologías usadas

- HTML5
- CSS3
- JavaScript (vanilla)

---

## Uso

1. Clona o descarga el repositorio.
2. Abre el archivo `index.html` en tu navegador preferido.
3. Lee el mensaje de cumpleaños.
4. Haz clic en el botón **"Abrir Sorpresa 2 🎁"** para jugar al jueguito de atrapar sushi.
5. Cada sushi que captures suma un punto.
